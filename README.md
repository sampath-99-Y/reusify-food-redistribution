# ♻️ Reusify - A Responsibility

> Turning excess into impact through intelligent food redistribution.

---

## 🌍 Overview

**Reusify** is a smart food redistribution platform that connects restaurants with surplus food to NGOs in high-demand areas using intelligent allocation and geospatial visualization.

The system ensures that excess food is efficiently redirected instead of wasted.

---

## 🚀 Key Features

- 🍱 Smart Food Allocation Algorithm  
- 🗺️ Interactive Map with Live Routing  
- 🔴 High-Demand Zone Highlighting  
- 📊 Real-Time Impact Metrics  
- 🔄 Dynamic Data Simulation (Refresh-based)  
- 🌐 Real-world restaurant & NGO data  

---

## 🧠 How It Works

1. Restaurants generate surplus food data  
2. NGOs provide demand levels  
3. System matches supply → demand intelligently  
4. Food routes are visualized on a live map  

---

## 🛠️ Tech Stack

- **Frontend & App:** Streamlit  
- **Visualization:** Plotly  
- **Backend Logic:** Python  
- **Data Handling:** Pandas
